# test-electron-nightmare

# How to #

Step 1

* npm run dist

Step 2
* npm run extract

Step 3

* Move the untransformed, electron-nightmare module package from the resources/appShadow/node_modules folder into the resources/app/node_modules folder

Step 4

* npm run pack
